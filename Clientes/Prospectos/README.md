# ETAPA 1 — PROSPECTOS
**Agente responsable:** 🔴 The Hunter  
**Descripción:** Resultados de cada sesión de búsqueda activa en Google Maps, redes y directorios.

---

## Cuándo se usa esta carpeta

Cada vez que The Hunter realiza una búsqueda de prospectos, crea una subcarpeta con la fecha y el sector prospectado. Aquí viven todos los negocios encontrados **antes** de ser contactados.

## Estructura esperada

```
Prospectos/
└── 2026-04-13_Sesion_01_Clinicas/
    ├── RESUMEN_SESION.md          ← índice de la sesión
    ├── ClinicaDentalPerez/
    │   └── PERFIL_PROSPECTO.md
    └── ClinicaSantaMaria/
        └── PERFIL_PROSPECTO.md
```

## Cómo avanzar un prospecto

Cuando se confirma primer contacto → mover su carpeta a `../En_Diagnostico/`
