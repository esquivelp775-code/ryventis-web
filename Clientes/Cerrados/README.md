# ETAPA 6 — CERRADOS
**Agente responsable:** ⚪ The Scribe  
**Descripción:** Relaciones finalizadas — tanto proyectos completados satisfactoriamente como prospectos que no cerraron.

---

## Cuándo se usa esta carpeta

Un registro llega aquí cuando:
- El cliente canceló el retainer (offboarding completado)
- El prospecto rechazó la propuesta definitivamente
- El proyecto terminó su ciclo natural sin retainer

## Estructura esperada

```
Cerrados/
└── ClinicaDentalPerez/
    ├── [archivos del historial completo]
    ├── LECCION_APRENDIDA.md        ← POST_VENTA-04 (obligatorio)
    └── MOTIVO_CIERRE.md            ← razón y fecha de cierre
```

## Nota importante

Un cliente cerrado en buenos términos puede regresar. The Bridge programa seguimiento a 90 días desde el offboarding. No eliminar carpetas — solo archivar aquí.
