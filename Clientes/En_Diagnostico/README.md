# ETAPA 2 — EN DIAGNÓSTICO
**Agente responsable:** 🔴 The Closer + 🟣 The Insight  
**Descripción:** Prospectos con los que ya hubo primer contacto confirmado y están en proceso de diagnóstico.

---

## Cuándo se usa esta carpeta

Un prospecto pasa aquí cuando:
- Respondió el primer mensaje y hay interés confirmado
- Se agendó o realizó la llamada de diagnóstico

## Documentos que debe contener la carpeta del negocio

```
En_Diagnostico/
└── ClinicaDentalPerez/
    ├── PERFIL_PROSPECTO.md         ← heredado de Prospectos/
    ├── BRIEFING_DESCUBRIMIENTO.md  ← COMERCIAL-01 completado
    └── AUDITORIA_FUGA_DINERO.md    ← COMERCIAL-02 (opcional en esta etapa)
```

## Cómo avanzar

Cuando se envía propuesta formal → mover carpeta a `../Propuesta_Enviada/`
