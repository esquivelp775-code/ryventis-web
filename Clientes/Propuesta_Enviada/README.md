# ETAPA 3 — PROPUESTA ENVIADA
**Agente responsable:** 🔴 The Closer  
**Descripción:** Prospectos que recibieron propuesta formal y están en proceso de decisión.

---

## Cuándo se usa esta carpeta

Un prospecto pasa aquí cuando:
- Se le envió la propuesta comercial (COMERCIAL-03 o COMERCIAL-07)
- Se está en fase de negociación o manejo de objeciones

## Documentos esperados

```
Propuesta_Enviada/
└── ClinicaDentalPerez/

    ├── BRIEFING_DESCUBRIMIENTO.md
    ├── PROPUESTA_2026-04-13.md     ← COMERCIAL-03 o 07 completado
    └── SEGUIMIENTO.md             ← registro de mensajes de follow-up (COMERCIAL-04)
```

## Cómo avanzar

Cuando firma contrato y The Treasurer confirma anticipo → mover a `../En_Curso/`

Si no cierra → mover a `../Cerrados/` con nota de razón
