## FILTRO DE CALIFICACIÓN — CRITERIO BANT+

*Responder cada criterio con Sí (✅), No (❌) o Parcial (🟡). Mínimo 3 ✅ para proceder.*

### B — BUDGET (Presupuesto)
```
¿Tiene capacidad de pago estimada para el servicio más básico ($4,000+ MXN)?
  ✅ Sí — negocio establecido con ingresos visibles
  🟡 Posible — negocio pequeño pero con dolor real
  ❌ No — negocio en etapa muy inicial o en crisis

Evidencia observada:
```

### A — AUTHORITY (Autoridad)
```
¿La persona de contacto toma decisiones de inversión?
  ✅ Sí — es el dueño o tiene poder de decisión confirmado
  🟡 Parcial — influye pero no decide solo
  ❌ No — es empleado sin poder de decisión

Evidencia observada:
```

### N — NEED (Necesidad)
```
¿Tiene un problema operativo claro que Ryventis puede resolver?
  ✅ Sí — dolor identificado, cuantificable, urgente
  🟡 Parcial — hay ineficiencia pero no está consciente del costo
  ❌ No — operación eficiente o no alineada con nuestros servicios

Problema principal identificado:
```

### T — TIMELINE (Tiempo)
```
¿Tiene urgencia o un horizonte de tiempo definido para resolver el problema?
  ✅ Sí — quiere resolverlo este mes o el próximo
  🟡 Parcial — "en algún momento" o "cuando haya presupuesto"
  ❌ No — no hay urgencia definida

Urgencia observada:
```

### + — FIT (Alineación con perfil ideal)
```
¿Es el tipo de cliente con quien Ryventis trabaja bien?
  ✅ Sí — PYME de servicios, Puebla, dueño receptivo a tecnología
  🟡 Parcial — fuera de zona o sector secundario
  ❌ No — perfil de cliente conflictivo, fuera de alcance o sector no prioritario

Observaciones:
```
