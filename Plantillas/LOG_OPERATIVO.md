```
  → LEGAL-03: Recibo del anticipo

  → LEGAL-06: Orden de Trabajo (señal verde oficial)
  → TECNICO-01: Blueprint de Arquitectura
  → TECNICO-06: Plan de Integraciones

DESARROLLO (The Stack):
  → TECNICO-03: Diccionario de Datos y APIs
  → TECNICO-02: Protocolo QA (antes de entregar)
  → TECNICO-04: Checklist Deploy a Producción

ENTREGA (The Bridge + The Stack):
  → POST_VENTA-01: Manual de Usuario
  → POST_VENTA-02: Video Script de Entrega
  → LEGAL-04: Acuerdo de Mantenimiento Retainer
  → LEGAL-03: Recibo del saldo final

POST-VENTA (The Bridge + The Scribe):
  → POST_VENTA-03: Reporte Mensual (cada mes)
  → POST_VENTA-05: Checkpoint 30 Días (al mes 1)
  → POST_VENTA-06: Upsell y Retención (cuando aplique)

  → TECNICO-05: Documentación Técnica (para The Scribe)
```
