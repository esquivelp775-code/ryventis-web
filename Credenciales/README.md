## Reglas de esta carpeta

1. **Nunca subir a GitHub** — esta carpeta debe estar en `.gitignore` si se usa control de versiones
2. **Nunca compartir por WhatsApp o correo** — solo acceso local
3. **Una subcarpeta por servicio/herramienta** — no mezclar credenciales en un solo archivo
4. **Actualizar inmediatamente** cuando cambies una contraseña o dato
5. **Formato estándar:** cada archivo sigue la plantilla `_PLANTILLA.md` de esta carpeta

## Cómo agregar nuevas credenciales

1. Crea una subcarpeta con el nombre del servicio (ej: `Supabase/`, `Vercel/`, `Twilio/`)
2. Copia `_PLANTILLA.md` dentro de esa subcarpeta
3. Renómbrala con el nombre del servicio (ej: `SUPABASE_PROD.md`)
4. Llena todos los campos
5. Actualiza el índice de esta tabla
