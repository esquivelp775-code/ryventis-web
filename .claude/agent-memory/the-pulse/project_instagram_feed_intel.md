Estilo de feed seleccionado para @ryventis_ia: **Columnas Verticales + Armonía de Marca Oscura** (Column Layout con color system oscuro cian/violeta).

Patron concreto: columna izquierda = contenido educativo/datos (graficas, stats), columna central = Reels cara a camara, columna derecha = quotes/prueba social. Fondo negro profundo (#0A0A0F) como base universal. Acentos cian (#00E5CC) en textos clave y bordes de tarjetas. Violeta (#7C3AED) como marcador de categoria.

**Why:** Decision tomada el 2026-04-23 por The Pulse para The Creative. El estilo de columnas es el unico que: (a) soporta mezcla de Reels + carruseles + estaticos sin romper la coherencia, (b) opera con templates repetibles (bajo tiempo de produccion), (c) proyecta autoridad y organizacion ante el buyer persona PYME que desconfia de la tecnologia.

**How to apply:** Cuando The Creative pida brief visual o sistema de templates para Instagram, este es el patron validado. No usar puzzle ni gran formato — demasiado tiempo de produccion. El feed NO debe verse "raw" o descuidado a pesar de la tendencia 2026 — Ryventis es marca nueva que necesita proyectar competencia primero.

**Tendencias 2026 registradas:**
- El feed perfectamente curado esta en declive; Mosseri impulsa contenido "raw" y autentico
- Para marcas nuevas B2B esto es una trampa: sin reputacion establecida, la imperfeccion se lee como incompetencia
- Los feeds oscuros con templates de marca son el equilibrio correcto: profesional pero no frio
- Carruseles + Reels tienen mayor peso algoritmico que posts estaticos solos
- Checkerboard y Column Layout son los estilos con mejor ratio esfuerzo/resultado para agencias de servicios
- Bordes/marcos en negro funcionan como separador visual elegante en feeds oscuros
